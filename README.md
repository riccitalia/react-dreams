## Vue Dreams

Created with [create-react-app](https://create-react-app.dev).

```bash
# Clone repository and enter it
git clone https://github.com/riccitalia/react-dreams.git
cd react-dreams

# Install dependencies
npm install

# Serve at localhost:3000
npm start
```
